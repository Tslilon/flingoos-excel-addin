Flingoos Excel Logger - Help

Getting Started

The Flingoos Excel Logger captures detailed semantic events from Excel and sends them to a local listener server for workflow tracing.

Setup

  1. Start the Python listener server on your local machine: 
    python src/excel_logger.py
  2. Open Excel and go to the Insert tab > My Add-ins > find the Flingoos Excel Logger
  3. Click "Start Logging" to begin capturing Excel events
  4. Perform your Excel tasks
  5. Click "Stop Logging" when done

Troubleshooting

  • Connection Issues: Ensure the Python listener server is running on your local machine at port 5555
  • Add-in Not Loading: Check that you're using the latest version of Excel
  • No Events Logged: Check your firewall settings to ensure the add-in can connect to localhost:5555
